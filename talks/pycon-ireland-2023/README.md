# NiceGUI — Inventing Python's Nicest UI Framework

Talk given by Falko Schindler at **PyCon Ireland 2023**.
[Watch on YouTube](https://www.youtube.com/watch?v=D5DnLgeObq4)

## Running

```bash
cd talks/pycon-ireland-2023
uv run main.py
```

Navigate with the arrow keys or using the navigation bar on the bottom.
Open `/notes` in another tab or on another device to view the slide notes.
